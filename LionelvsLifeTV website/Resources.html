Resources

Welcome to the 'Resources' Page. Here you can find all the resources that I've used or am currently using in order to learn the things I'm interested in. Whether that's learning Japanese, how to program/code or how to do a handstand push up. You can find it all here.

Japanese

  • My Japanese Language Playlist - An AMAZING, well-structured, detailed playlist that teaches you how to study Japanese to fluency, by yourself, for FREE. Show's you how to use all the resources below.
  • Anki - A Spaced Repetition Software(SRS) that I use to memorise almost everything
  • All Japanese All The Time(AJATT) - Foundational theory and method for Learning Japanese
  • Refold - Foundational theory and method to learn any Language
  • Netflix - A Spaced Repetition Software(SRS) that I use to memorise almost everything
  • Animelon - A website to watch anime for free. Allow's you to watch anime with Japanese or English subtitles to aid in your studies.
  • Jisho - A Japanese-English Dictionary
  • DeepL - An amazing language translator. (Better than google translate in my humble opinion)
  • Ganma - A website where you can read Japanese Manga(IN JAPANESE) for free!

Programming

  • W3schools - A website where you can learn various programming language for free
  • Stackoverflow - A website where you can get help if you're ever stuck with anything coding related.
